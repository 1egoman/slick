# Reaction

Type: Native (built into slick)

Arguments:
- `<reaction name>` - name of emoji to react to message with.

Command aliases:
- `reaction`
- `react`
- `r`

## Description
React to the current message with a given emoji.
a
## Example

![gifs/Reaction.gif](gifs/Reaction.gif)

```lua
Reaction("smile")
```

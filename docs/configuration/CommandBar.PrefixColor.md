# CommandBar.PrefixColor

- Type: `color`
- Default: `::` [(format explanation)](../colors.md)

This configuration option specifies the style used to render the [command bar prefix](../CommandBar.md).

## Usage
`:set CommandBar.PrefixColor red:green:`
